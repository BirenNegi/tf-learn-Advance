# Terraform Learn – 30‑day interactive course

## Features
- 30 days of Theory, Lab, Challenge
- Embedded terminal (xterm.js + Docker sandbox with Terraform & Azure CLI)
- AI Deep Dive (Anthropic Claude)
- Modern dashboard

## Setup
1. Install Docker and Node.js
2. Run `npm install` in the main folder
3. Run `cd terminal-server && npm install && node index.js` (starts WebSocket backend)
4. In another terminal: `node server.js`
5. Open http://localhost:3000

## Environment Variables
- `ANTHROPIC_API_KEY` (optional, for AI Deep Dive)

Enjoy!
